💰 Expense Tracker
A modern web-based Expense Tracker application that helps users manage their finances efficiently by tracking income, expenses, and budgets in one place.

🔗 Live Demo: https://expense-tracker-seven-peach-64.vercel.app/

🚀 Features :-
📊 Expense Tracking – Add, edit, and delete transactions easily

💵 Income Management – Track multiple income sources

📂 Category-wise Organization – Classify expenses (Food, Transport, etc.)

📈 Visual Analytics – Interactive charts to analyze spending habits

🎯 Budget Management – Set limits and monitor usage

🔍 Transaction History – View all past records with filtering

📱 Responsive UI – Works across desktop and mobile devices

🔒 Secure Data Handling – Ensures safe storage of financial data


These features help users understand spending patterns and improve financial decision-making

⚙️ Installation & Setup :--

Clone the repository:
git clone https://github.com/your-username/expense-tracker.git

Navigate to the project folder:
cd expense-tracker

Install dependencies:
npm install

Run the development server:
npm run dev
Open in browser:
http://localhost:5173
